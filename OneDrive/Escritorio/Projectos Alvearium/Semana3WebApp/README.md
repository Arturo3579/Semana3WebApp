## 📱 ¿Qué es esto?

Básicamente, es mi primera aplicación web moderna hecha con React. Tiene varias páginas entre las que puedes navegar sin que se recargue el navegador (gracias a React Router), y ya está publicada en internet para que cualquiera pueda verla.

La idea era aprender a:
- Crear una app desde cero
- Dividirla en componentes
- Hacer que las cosas cambien cuando el usuario hace clic
- Y publicarla para que se pueda ver online



## 🗂 ¿Cómo está organizado el proyecto?

Así más o menos se ven las carpetas:

**src/
├── pages/ ← Aquí están las páginas principales
│ ├── HomePage.jsx (la portada)
│ ├── ProjectPage.jsx (el proyecto de la Semana 2)
│ ├── SecondPage.jsx (página de demo)
│ ├── AboutPage.jsx (sobre mí)
│ └── ResourcesPage.jsx (enlaces útiles)
├── components/ ← Componentes que se pueden reutilizar
├── assets/ ← Imágenes y cosas así
├── App.jsx ← Donde configuro las rutas
├── main.jsx ← El punto de entrada
└── index.css ← Estilos generales**
